# 배경 이미지 폴더

이 폴더에는 게임에서 사용되는 배경 이미지들을 저장합니다.

## 파일 명명 규칙

- `menu_bg.png` - 메뉴 화면 배경
- `game_bg.png` - 게임 플레이 화면 배경
- `result_bg.png` - 결과 화면 배경

## 권장 사양

- **해상도**: 1200x800 이상
- **형식**: PNG, JPG
- **용량**: 5MB 이하 권장

## 사용 예시

배경 이미지는 `ImageLoader` 클래스를 통해 자동으로 로드됩니다.
파일이 없을 경우 기본 그라디언트 배경이 사용됩니다.
